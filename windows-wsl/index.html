# Setup the development env under Windows WSL

# 开启Windows 10下的WSL2

Windows Subsystem for Linux Installation Guide for Windows 10 (opens new window)

# Enable the Windows Subsystem for Linux

dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

# Enable Virtual Machine feature

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

# 更新到WSL2

To update to WSL 2, you must be running Windows 10. Download the Linux kernel update packageWSL2 Linux kernel update package for x64 machines (opens new window)

# Set WSL 2 as your default version

wsl --set-default-version 2
